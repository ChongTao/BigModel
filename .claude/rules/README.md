# .claude/rules - 规则按需加载目录

本目录存放 BigModel 项目的各类规范和检查规则。系统会根据需要动态加载相关规则，保持主 `CLAUDE.md` 精简。

## 规则清单

### document-security.md
- **应用场景**：阅读或修改任何项目文档
- **检查内容**：敏感信息扫描（API Key、密码、私钥、内网 IP、个人信息等）
- **触发时机**：自动加载，不需用户手动启用

### git-workflow.md
- **应用场景**：提交代码或文档前
- **检查内容**：提交前必须通过敏感信息检查 hook
- **触发时机**：执行 `git commit` 前自动应用

### 其他规则
可根据需要添加更多规则文件，如：
- `code-review.md` - 代码审查规范
- `git-workflow.md` - Git 工作流规范
- `testing.md` - 测试规范

## 使用方式

规则系统设计为自动加载和按需加载的混合模式：
- **常驻规则**（如文档安全检查）：自动加载
- **工作流规则**（如 Git 工作流）：在相关场景自动应用
- **专项规则**：可在 CLAUDE.md 中显式引用

通过这种方式，保持 CLAUDE.md 清晰有序，同时支持灵活扩展。
