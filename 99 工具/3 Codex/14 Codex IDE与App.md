# 14 Codex IDE与App

Codex 除了 CLI，还可以在 IDE 和 App 中使用。

## 14.1 IDE 扩展

IDE 扩展适合：

- 直接围绕当前文件协作
- 利用打开文件和选中代码作为上下文
- 在编辑器内边聊边改

它和 CLI 共享配置，因此很多设置不需要在 IDE 里单独维护一套。

## 14.2 App

Codex App 更像桌面工作台，适合：

- 并行线程
- 跟踪计划、来源和任务摘要
- 管理 worktree、自动化和 Git 工作流
- 配合本地项目持续运行

## 14.3 什么时候用哪一个

用 CLI：

- 终端驱动开发
- shell、git、脚本协作多

用 IDE：

- 围绕当前代码片段快速协作
- 更强调编辑器上下文

用 App：

- 任务并行
- 持续跟踪多个线程
- 想把本地协作和桌面工作台结合起来
