# 2 Codex常见操作

## 2.1 启动 Codex

```sh
codex
```

进入交互式会话后，Codex 会在当前目录读取项目上下文、规则文件和配置。

## 2.2 在当前项目里协作

常见提示方式：

- 先让它理解目录结构
- 明确目标文件或模块
- 指定是否可以改文件、跑命令、写测试

例如：

```text
先阅读这个仓库的结构，说明入口、核心模块和数据流。
```

```text
修复当前分支里的这个 bug，只修改最小范围代码，并补充必要验证。
```

## 2.3 查看当前规则是否生效

如果仓库里有 `AGENTS.md`，可以让 Codex 先总结当前加载到的规则：

```text
先总结当前会话已经加载的项目规则，再开始改动。
```

## 2.4 常见工作流

### 阅读代码

```text
先不要改代码，说明这个模块的职责、关键入口和依赖关系。
```

### 实现功能

```text
实现这个需求，先给出最小改动方案，再直接修改代码并验证。
```

### 做代码审查

```text
审查当前分支相对主分支的变化，重点看 bug、回归风险和测试缺口。
```

### 整理文档

```text
按当前目录风格拆分文档，并更新 README 入口。
```

## 2.5 交互原则

想让 Codex 稳定一些，通常有几个实用原则：

- 先说目标，再说限制
- 明确它能不能改文件、跑命令、联网
- 大任务先拆块
- 让它在动手前先总结现状
- 改完后要求它做最小验证

## 2.6 什么时候该用 Subagents

如果任务同时包含多条可并行子线，例如：

- 多个文件独立分析
- 多个 review 维度并行检查
- 多个模块独立调研

这时更适合显式让 Codex 使用子代理，而不是单线程硬做。

## 2.7 CLI 基础命令

常见命令：

| 命令 | 说明 |
| :--- | :--- |
| `codex` | 启动交互式会话 |
| `codex exec "任务"` | 非交互模式执行单次任务 |
| `codex login` | 登录或切换认证方式 |
| `codex --profile review` | 使用指定 profile 启动 |
| `codex --help` | 查看帮助 |
| `codex --version` | 查看版本 |

## 2.8 Slash Commands

Codex 在交互会话里支持 `/` 开头的命令。不同环境下可用命令会有差异，最稳妥的方式是直接输入 `/` 查看当前列表。

当前常见命令包括：

| 命令 | 作用 |
| :--- | :--- |
| `/model` | 切换模型 |
| `/fast` | 调整更快的会话模式 |
| `/personality` | 调整回答风格 |
| `/permissions` | 调整权限与审批策略 |
| `/approve` | 处理待审批操作 |
| `/agent` | 切换或查看子代理线程 |
| `/status` | 查看当前会话状态 |
| `/compact` | 压缩上下文 |
| `/diff` | 查看当前 Git 变更 |
| `/plugins` | 查看和管理插件 |

补充说明：

- 任务执行中输入 slash command 后，可以排队到下一轮执行
- 一些命令只在特定环境出现，例如 IDE 上下文相关命令或 Windows 的沙箱读目录命令
- 不要硬背命令总数，以当前会话里的 `/` 列表为准

## 2.9 AGENTS.md 层级

`AGENTS.md` 支持分层加载，越靠近当前工作目录的文件，优先级通常越高。

常见层级：

| 层级 | 路径 | 作用范围 |
| :--- | :--- | :--- |
| 全局层 | `~/.codex/AGENTS.md` | 跨项目通用约定 |
| 全局覆写层 | `~/.codex/AGENTS.override.md` | 临时替换全局规则 |
| 仓库层 | `repo/AGENTS.md` | 仓库级规范 |
| 子目录层 | `repo/subdir/AGENTS.md` 或 `AGENTS.override.md` | 子目录特殊规则 |

同一目录下如果存在 `AGENTS.override.md`，则优先使用它，而不是同级 `AGENTS.md`。

## 2.10 配置文件路径

常见位置：

| 文件 | 路径 | 作用 |
| :--- | :--- | :--- |
| 用户配置 | `~/.codex/config.toml` | 全局默认配置 |
| 项目配置 | `.codex/config.toml` | 项目特定配置 |
| 项目指令 | `AGENTS.md` | 项目行为规范 |
| 用户技能 | `~/.codex/skills/` | 本地 Skills |
| 项目插件 | `.codex/plugins/` | 项目插件或相关配置 |

## 2.11 快捷键

CLI 里常见快捷键：

| 快捷键 | 功能 |
| :--- | :--- |
| `Enter` | 发送消息 |
| `Shift+Enter` | 换行 |
| `Ctrl+C` | 中断操作 |
| `Ctrl+C` 两次 | 退出 Codex |
| `Ctrl+O` | 复制最后回复 |

不同终端和平台可能存在差异，具体以当前客户端实际表现为准。
